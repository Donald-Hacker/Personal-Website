Portfolio

Howdy Hack Website

A website that included information about Howdy Hack including the ability to register as well. The purpose was to create a less buggy and more customized website that fitted the needs of the organization running the event as oppose to using an open-source registration website. This included a more streamlined registration system as well as a more organic design for the website.

Menu-Driven Data Regression Calculator

A MATLAB program using a graphical user interface that instructs the user to insert a text file containing two columns of data. The first column corresponds to the x-axis. The second column corresponds to the y-axis. The program would then analyze the text file given and offer the user a menu to guide them. Options for that menu included a selection of regression model, graph labeling, and exporting of the graph into a PNG file. The graph also included information such as mean square error and the regression equation.

Arduino Puzzle Solver

An Arduino Robot programmed in C++ with the ability to solve a maze and an obstacle course and arrange a set of blocks together. In this project, the Arduino robot was designed using three different programs. Each program used various sensors located throughout the robot that allowed it to detect distance, color, and light to accomplish each task.
